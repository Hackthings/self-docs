Detailed Description

This abstract interface defines an interface for storing data and finding that data by indexed fields. Facade design pattern is being used here.

IDataStore * IDataStore::Create ( const std::string &  a_DB,
const Json::Value &  a_Definition 
)
static

Create a data store object using the provided definition. Definitions are structured JSON data that describe the data that will be stored in this store. We look for a named json at the root called "Indexed_" which is an array of all data that should be indexed by the store for the Find() function. Example Definition: { "_Indexed" : [ "ID", "response/text" ], // index the ID & text data "ID": "", "response": { "text": "Hello World" } }
