Detailed Description

The BlackBoard class acts as the central publish/subscribe system for all agents, classifiers, and extractors.

Member Function Documentation

◆ AddThing()

void BlackBoard::AddThing ( const IThing::SP &  a_spThing)

Add a concept to this BlackBoard, this concept will automatically be connected to other concepts to produce a Goal in the end. This object takes ownership of the provided object and will delete the concept once a goal is produced.

◆ OnThingEvent()

Invoke this function to pass a event to all subscribers for a given object type. This is invoked automatically by AddThing(), RemoveThing(), and when a state is changed on a IThing object.
